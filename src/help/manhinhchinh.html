Bạn có thể sử dụng các chức năng trên phần mềm bằng cách click vào các nút phía trên ảnh nền như: Đăng xuất Kết thúc chuyên đề Người học Khóa học Học viên Hướng dẫn

Menu hệ thống

Ngoài ra trên các thanh menu cũng có nhiều tùy chọn hơn ví dụ:
muốn đổi mật khẩu: chọn menu hệ thống -> Đổi mật khẩu
muốn Đăng nhập: chọn menu hệ thống -> Đăng nhập hoặc phím tắt CTRL+L
muốn Đăng xuất: chọn menu hệ thống -> Đăng xuất hoặc phím tắt CTRL+O
muốn kết thúc: chọn menu hệ thống -> Kết thúc hoặc phím tắt F10

Menu Quản lý

quản lý chuyên đề: chọn menu quản lý -> chuyên đề hoặc phím tắt ctrl+f1
quản lý người học: chọn menu quản lý -> người học hoặc phím tắt ctrl+f3
quản lý khóa học: chọn menu quản lý -> khóa học hoặc phím tắt ctrl+f2
quản lý học viên: chọn menu quản lý -> học viên hoặc phím tắt ctrl+f4
quản lý nhân viên: chọn menu quản lý -> nhân viên hoặc phím tắt ctrl+f5

Menu thống kê

Thống kê bảng điểm: thống kê -> Bảng điểm hoặc phím tắt Shift+F1
Thống kê lượng người học: thống kê -> lượng người học hoặc phím tắt Shift+F2
Thống kê điểm chuyên đề: thống kê -> điểm chuyên đề hoặc phím tắt Shift+F3
Thống kê doanh thu: thống kê -> doanh thu hoặc phím tắt Shift+F4

Menu Trợ giúp

Hướng dẫn sử dụng: Trợ giúp -> Hướng dẫn sử dụng hoặc phím tắt F1
giới thiệu sản phẩm: Giới thiệu sản phẩm
